# Haushaltsbuch

Ein kleines sehr schlankes Haushaltsbuch

Es ist ein kleines Haushaltsbuch auf Basis von PHP und PDO
Mit Absicht sehr simpel gehalten und intuitiv zu bedienen

Du benötigst einen Webspace mit PHP 8.1 und höher und eine MySQL Datenbank

# Installation:
In der db.php trägst du die Datenbankdaten ein
Lädst dann alle Dateien auf den Webspace hoch
Nun rufst du die install.php im Browser auf, um die Datenbanktabellen anzulegen
Nun kannst du loslegen.
Beim ersten Aufruf wirst du gezwungen einen Nutzer anzulegen mit Email und Passwort.

Wünsche und Anregungen sind Willkommen
